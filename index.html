TAREA N°1

1.¿Qué es un Protocolos Http?

El Http es un protocolo de transferencia sobre el que se basa la red informática mundial(www),
nos permite realizar una petición de datos y recursos.

2.¿Qué es un Query String?

Es un término que se utiliza para hacer referencia a una interacción con una base de datos.
Además, es la parte de una URL que contiene los datos que deben pasar a las aplicaciones web.

3.¿Qué es un Verbos HTTP?

Los verbos HTTP definen la acción que se realizará sobre un determinado recurso.
con ello,Http involucra una cantidad de metodos que son GET, POST, PUT, PATCH y DELETE.

4.¿Qué es un DNS?

Es una tecnología basada en una base de datos que sirve para resolver nombres en las redes, es decir,
para conocer la dirección IP de la máquina donde está alojado el dominio al que queremos acceder.

5.¿Qué es un CDN?

Red de servidores que distribuye contenido de un servidor de "origen" a todo el mundo.
